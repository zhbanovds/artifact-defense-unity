# Учебный проект "Программирование игр"

## Жбанов Д.С.
## Группа М8О-401Б-22

Artifact Defense — учебный проект на Unity в жанре action / arena defense. Игрок управляет героем ближнего боя и защищает магический кристалл от волн врагов.

## Демонстрация проекта

Видео с демонстрацией игрового процесса доступно по ссылке:

[Смотреть демонстрацию на Google Drive](https://drive.google.com/drive/u/0/my-drive)

## Описание проекта

Цель игры — не дать монстрам уничтожить кристалл. Враги появляются волнами, двигаются к кристаллу и атакуют его. Игрок должен уничтожать противников с помощью атак и специальных способностей.

Особенность проекта заключается в том, что у игрока нет собственной полоски здоровья: главным объектом защиты является кристалл.

## Основные задачи проекта

- реализовать управление персонажем;
- добавить систему атак ближнего боя;
- реализовать движение врагов к кристаллу;
- создать систему волн противников;
- добавить здоровье кристалла;
- реализовать условия победы и поражения;
- создать базовый пользовательский интерфейс;
- использовать визуальные эффекты и ассеты Unity.

## Реализованные механики

- управление главным персонажем;
- обычная атака и альтернативная атака;
- волны врагов;
- несколько типов противников;
- здоровье кристалла;
- накопление энергии кристалла;
- специальная массовая способность;
- защитное поле кристалла;
- экран победы;
- экран поражения;
- главное меню;
- игровой интерфейс.

## Управление

| Действие | Клавиша / кнопка |
|---|---|
| Перемещение | WASD |
| Обычная атака | Левая кнопка мыши |
| Альтернативная атака | Правая кнопка мыши |
| Специальная способность | E |
| Защитное поле | Q |
